Pruebas para el parámetro firstName al crear un/a usuario/a

Requisitos previos

- Tener instalados los paquetes pytest y requests para ejecutar las pruebas.

Ejecución de pruebas

- Ejecuta todas las pruebas con el comando: pytest.

Descripción de pruebas

Este conjunto de pruebas verifica el comportamiento del parámetro firstName al crear un/a usuario/a en [nombre de la API/endpoint]. Las pruebas cubren diferentes escenarios, incluyendo:

- Valores válidos y válidos con longitud límite.
- Valores inválidos (caracteres especiales, números, espacios).
- Valores con longitud mayor o menor a la permitida.
- Ausencia del parámetro.

Resultados esperados

- Cada prueba debe retornar un código de estado HTTP correspondiente al resultado esperado.

Información adicional

- Para reportar errores o sugerencias, por favor abrir un issue en este repositorio.
